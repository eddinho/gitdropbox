# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-02-17
- Rewrote `gitdropbox` with strict error handling and safer CLI parsing.
- Added support for `--message`, `--update-remote`, `--no-color`, and `--version`.
- Added validation for branch names, remote names, and bare repository integrity.
- Added environment-variable and `git config` defaults documentation.
- Added CI workflow with ShellCheck and end-to-end smoke test.
- Added `CONTRIBUTING.md` and `RELEASE_CHECKLIST.md` for publishing workflow.
